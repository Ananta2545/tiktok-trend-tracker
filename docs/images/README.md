# Image Assets

This directory contains screenshots and visual assets for the README.

## Required Images

- `dashboard.png` - Main dashboard screenshot
- `charts.png` - Trend charts and analytics view
- `api-dashboard.png` - API rate limit dashboard
- `alerts.png` - Alert management interface
- `video-thumbnail.png` - Demo video thumbnail

## Image Guidelines

- **Format**: PNG or JPG
- **Resolution**: 1920x1080 (Full HD) or higher
- **File Size**: < 2MB per image
- **Naming**: Use kebab-case (lowercase with hyphens)

## How to Add Images

1. Take screenshots of the application
2. Save them in this directory
3. Update the README.md with actual image links
4. Uncomment the image tags in README.md

Example:
```markdown
![Dashboard Screenshot](docs/images/dashboard.png)
```
